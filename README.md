# 📚 Course Chatbot

This is a simple chatbot created for a course project.  
It runs directly in the browser using HTML, CSS, and JavaScript.

## 🚀 Features
- Simple question-answer chatbot
- Built with JavaScript (no backend needed)
- Runs entirely in the browser
- Can be embedded in Moodle or other platforms

## 🌐 Live Demo
[Click here to open the chatbot](https://foteinikara.github.io/MAY341-course-chatbot/)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📦 How it works
The chatbot uses simple rule-based logic to respond to user input (keywords like "exam", "lecture", etc.).
